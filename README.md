# Timeline Trace

Instruments and visualizes execution of JS code. Supports detailed
tracking of:

* Values of each variable
* Function calls
* Loops
* Conditions

# Setup

```
npm install
npm run build
npm run watch
```

This project does not provide a web server, but you could use one
such as:

```
python -m SimpleHTTPServer
```

# Testing

```
npm run test
```
